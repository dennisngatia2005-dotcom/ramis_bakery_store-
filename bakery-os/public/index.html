Flask Backend API Documentation
Base URL & Setup

Base URL: http://localhost:5000/api

All requests use Content-Type: application/json

All responses return { "status": "ok", "data": {...} } or { "status": "error", "message": "..." }

Production Endpoints

POST /api/production/log Log a crate completion

Request body:

{
  "baker_name": "Jane Mwangi",       // string, required
  "baker_id": "BKR-001",          // string, required
  "crates_completed": 5,        // integer, required
  "completed_at": "2024-01-15T10:30:00", // ISO datetime, required
  "notes": "Morning batch"       // string, optional
}

Response:

{
  "status": "ok",
  "data": { "id": 1, "message": "Production logged" }
}
GET /api/production/logs Get all production logs

Query params: ?date=2024-01-15 (optional filter by date)

{
  "status": "ok",
  "data": [
    {
      "id": 1,
      "baker_name": "Jane Mwangi",
      "baker_id": "BKR-001",
      "crates_completed": 5,
      "completed_at": "2024-01-15T10:30:00",
      "notes": "Morning batch"
    }
  ]
}

Transport Endpoints

POST /api/transport/trip Start a new delivery trip 
{
  "driver_name": "Peter Kamau",    // string, required
  "driver_id": "DRV-001",         // string, required
  "crates_dispatched": 20,       // integer, required
  "departed_at": "2024-01-15T08:00:00" // ISO datetime, required
}

Response returns trip_id for later use.

POST /api/transport/trip/:id/arrive Mark trip as arrived 
{
  "arrived_at": "2024-01-15T10:30:00",  // ISO datetime, required
  "crates_returned": 3               // integer, required (unsold crates)
}

Backend auto-calculates duration_minutes from departure to arrival.

GET /api/transport/trips Get all trips

Query params: ?status=active or ?status=completed

{
  "status": "ok",
  "data": [
    {
      "id": 1,
      "driver_name": "Peter Kamau",
      "crates_dispatched": 20,
      "crates_returned": 3,
      "departed_at": "2024-01-15T08:00:00",
      "arrived_at": "2024-01-15T10:30:00",
      "duration_minutes": 150,
      "status": "completed"
    }
  ]
}

Sales Endpoints

POST /api/sales/record Record a sale 
{
  "seller_name": "Mary Njeri",      // string, required
  "seller_id": "SLR-001",          // string, required
  "sale_type": "wholesale",        // "wholesale" | "retail"
  "crates_sold": 10,               // integer, required
  "price_per_crate": 500,          // number, required (KES)
  "total_amount": 5000,            // number, required (KES)
  "payment_due": "2024-01-20",     // date string, required
  "payment_status": "pending",     // "paid" | "pending"
  "mpesa_ref": "QK2X1P9ABC"       // string, optional (M-Pesa ref)
}
POST /api/sales/mpesa/callback M-Pesa STK Push callback

This endpoint receives callbacks from Safaricom Daraja API after STK Push payment.

{
  // Sent BY Safaricom to your server:
  "Body": {
    "stkCallback": {
      "MerchantRequestID": "...",
      "CheckoutRequestID": "...",
      "ResultCode": 0,          // 0 = success
      "ResultDesc": "The service request processed successfully",
      "CallbackMetadata": {
        "Item": [
          { "Name": "Amount", "Value": 5000 },
          { "Name": "MpesaReceiptNumber", "Value": "QK2X1P9ABC" },
          { "Name": "PhoneNumber", "Value": 254712345678 }
        ]
      }
    }
  }
}

⚠ Register this URL on Safaricom Daraja portal. Must be HTTPS in production.

GET /api/sales/seller/:id Look up seller by name or ID

URL param: seller ID or use query ?name=Mary for name search

{
  "status": "ok",
  "data": {
    "seller_id": "SLR-001",
    "seller_name": "Mary Njeri",
    "total_sales": 12,
    "pending_payments": 2,
    "recent_sales": [...]
  }
}

    "transport": { "active_trips": 2, "completed_today": 3 },
    "sales": { "revenue_today": 45000, "pending_payments": 4 }
  }
}
Flask SQLite Schema (Copy This)
# models.py - paste this into your Flask app

CREATE TABLE production_logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    baker_name TEXT NOT NULL,
    baker_id TEXT NOT NULL,
    crates_completed INTEGER NOT NULL,
    completed_at DATETIME NOT NULL,
    notes TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE transport_trips (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    driver_name TEXT NOT NULL,
    driver_id TEXT NOT NULL,
    crates_dispatched INTEGER NOT NULL,
    crates_returned INTEGER DEFAULT 0,
    departed_at DATETIME NOT NULL,
    arrived_at DATETIME,
    duration_minutes INTEGER,
    status TEXT DEFAULT 'active',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE sales_records (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    seller_name TEXT NOT NULL,
    seller_id TEXT NOT NULL,
    sale_type TEXT NOT NULL,
    crates_sold INTEGER NOT NULL,
    price_per_crate REAL NOT NULL,
    total_amount REAL NOT NULL,
    payment_due DATE NOT NULL,
    payment_status TEXT DEFAULT 'pending',
    mpesa_ref TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
